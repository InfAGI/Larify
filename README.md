# Larify

一个持续学习的 AI 嵌入式工程师。

## 特性

- **全流程开发** —— 给定一个目标，自主在真实硬件上走通整个开发流程。
- **持续积累经验** —— 主动探索、动手验证，把每次任务的经验沉淀下来。

## 安装

前置：本机已装 [Node.js](https://nodejs.org/)。

在 Claude Code 中依次执行三条命令。

添加插件市场：

```
/plugin marketplace add InfAGI/Larify
```

安装插件：

```
/plugin install Larify@Larify-marketplace
```

重载插件：

```
/reload-plugins
```
